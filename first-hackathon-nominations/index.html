Nominations for the First BOSS Hack-A-Thon

Nov 27th, 2012

The following projects were nominated as possible projects for our first hackathon on 2013-01-26

  • OpenStreeMap TerpNav

    Hacking on TerpNav from the University of Maryland

  • Shared Memory Shenanigans

    Take the concepts from the sylvester gem by Jeremy Holland and produce something dangerous and/or generally useful

  • Magic Gathering Card Sorterator

    Develop software to recognize cards from Magic: The Gathering for tournaments. Apparently these tournaments have to catalog the cards you got and the cards you’re going to play. Other possibilities are to build a sorting machine like a money counter to count these cards

  • Linux AirPlay-esque service

    Josh Adams and Adam Gamble talked about writing an AirPlay -esque service for linux (but it should be easy to run it on mac / windows as well) just having the server script vlc and having the host encode the desktop to a streaming video, and have a protocol to sync them up. They think it’s doable in the time allotted with a couple or three people working on it.
